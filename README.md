# Лендинг репетитора по математике — Дмитрий (AlgoMath)

Одностраничный сайт-визитка репетитора по математике для 5–10 классов, г. Бельцы.
Готовый статический HTML-файл, без сборки и зависимостей — открывается сразу в браузере.

## Файл

- `dmitry-tutor-landing-premium.html` — основной, актуальный вариант дизайна (строгий, премиальный: navy/графит/крем + изумрудный акцент).
- `dmitry-tutor-landing.html` — предыдущая версия (яркая, оранжевая с клетчатой текстурой). Оставлена для истории/сравнения, можно удалить.

## Стек

- Чистый **HTML + CSS**, без React и сборщиков.
- **Tailwind CSS** подключён через CDN (`cdn.tailwindcss.com`) — классы работают прямо в разметке.
- Шрифты **Manrope** (заголовки) и **Inter** (текст) — подключены через Google Fonts.
- Немного **vanilla JS** в конце файла — только для плавного появления блоков при скролле (`IntersectionObserver`).

Интернет нужен только для загрузки шрифтов и Tailwind-скрипта — сама разметка работает офлайн.

## Структура секций

1. **Шапка** — логотип/имя, навигация по якорям, кнопка «Записаться»
2. **Hero** — заголовок, оффер, цена (100 лей/час), карточка с условиями
3. **Обо мне** (`#about`) — текст + место под фото
4. **Услуги** (`#services`) — 4 карточки: ДЗ, пропущенные темы, подготовка к экзаменам, индивидуальный подход
5. **Условия и оплата** (`#terms`) — цена, пакеты занятий, кнопка «Оплатить онлайн»
6. **Футер / Контакты** (`#contacts`) — Viber/Telegram, Telegram-канал, e-mail

## Как редактировать

Файл — обычный HTML, весь текст можно менять прямо в разметке через поиск нужной фразы. Цвета вынесены в CSS-переменные в начале файла (`:root { --navy: ...; --emerald: ...; }`) — чтобы поменять палитру, достаточно поменять значения там, а не искать по всему файлу.

## Как открыть / разместить

- **Посмотреть локально**: просто открыть `.html`-файл в браузере (двойной клик).
- **Опубликовать**: файл можно загрузить как есть на любой статический хостинг (GitHub Pages, Netlify, Vercel, обычный хостинг с загрузкой файлов) — назвать `index.html` и разместить в корне сайта.
